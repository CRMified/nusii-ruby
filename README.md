# Nusii

Ruby library for the Nusii API.

## Installation

To get the latest stable, add this to your Gemfile:

    gem 'nusii'

To get the latest code, add this:

    gem 'nusii', git: https://github.com/Nusii/nusii-ruby

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nusii
